Welcome to my simple HTML portfolio. Use the links above to view my Glitch images and Meme images pages.

How to add your images

  1. Create a folder named images in the same folder as these HTML files.
  2. Put your image files in the images folder, for example:
    • images/glitch1.jpg
    • images/glitch2.png
    • images/meme1.jpg
  3. Edit glitch.html and meme.html to match your actual filenames if they differ.

If you prefer external hosting, replace the img src with a full URL (for example: https://i.imgur.com/yourimage.jpg).

Quick tips

  • Use lowercase filenames with no spaces (e.g., glitch1.jpg)
  • Prefer .jpg or .png
  • Preview locally with VS Code + Live Server (see below)
